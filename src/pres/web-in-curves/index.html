SVG — Scalable Vector Graphics
Масштабируемая Векторная Графика.

Развитие

  • SVG 0.0 → 1998 г.
  • SVG 1.0 → 2001 г.
  • SVG 1.1 → 2003 г.
  • SVG 1.2 → ∞
  • SVG 2.0 → когда-нибудь…

Застой с SVG 1.2 преодолён, ожидаем версию 2.0.

Профили

  • SVG Tiny → для мобильников
  • SVG Basic → для смартфонов

Профили появились в SVG 1.1.

Особенности

  • Векторный, а не растровый → масштабируется
  • Текстовый, а не бинарный → доступен *
  • Хитрый, а не как GIF → стили, скрипты, анимация…

31 августа Google заявил о начале индексации SVG.

Анимация

Анимация

SMIL → Synchronized Multimedia Integration Language *

  • SMIL 1.0 → 1998 г.
  • SMIL 2.1 → 2005 г.
  • SMIL 3.0 → 2008 г.

Произносится как «смайл».

Спиннер

			<rect width="50" height="150"
			    rx="25" ry="25" id="rect"/>
		
			<use x="225" y="0" xlink:href="#rect"/>

			</filter>

Inhuman UI

  • All interactive elements have an active state while pressed or focused from keyboard
  • All interactive elements have text labels.

Inhuman?

Not Accessible

Not Accessible

  • Broken mouse, small screen, slow network.
  • Small text and blocked viewport.
  • Scripts instead of links.
  • Fields that block pasting or autofill.
  • Bright sun, low contrast colors.
  • Wrong size or orientation of the screen.
  • Physical difficulties.

Labels

Labels

  • Content of an element: <a href>Home</a>
  • Caption of the group: <legend>Address</legend>
  • Attached element: <label for="input">Name</label>
  • Hidden label: <span class="visually-hidden">
  • Added label: <nav aria-label="Contents">

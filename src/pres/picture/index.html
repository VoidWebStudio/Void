			       src="400.png"

Брейкпоинт 400px

Меньше 600 пикселей.

  1. Весь экран: 400px → 400.png
  2. Весь экран 2x: 800px → 800.png
  3. Весь экран 3x: 1200px → 1200.png

Брейкпоинт 600px

Равен 600 пикселям.

  1. Полэкрана: 300px → 400.png
  2. Полэкрана 2x: 600px → 800.png
  3. Полэкрана 3x: 1200px → 1200.png

Брейкпоинт 800px

Больше 600 пикселей.

  1. Полэкрана: 400px → 400.png
  2. Полэкрана 2x: 800px → 800.png
  3. Полэкрана 3x: 1200px → 1200.png

Браузеры меньшие

Браузеры меньшие

  1. Отправляют адрес на прокси-сервер
  2. Прокси отрисовывает и отправляет результат обратно
  3. Результат отображается до следующего действия
  • Не работает интерактивный JavaScript
  • Не подгружаются веб-шрифты
  • Не работает CSS-графика
